# Project-01-TeamA
This project was bootstrapped with Fastify-CLI.

# 개발환경
Fastify-cli node v16 npm mongodb

# 기능명세

- 사용자는 투표 목록을 조회할 수 있다.
- 사용자는 특정 투표를 조회할 수 있다.
- 사용자는 투표를 할 수 있다.
- 사용자는 투표 결과를 확인할 수 있다.
- 관리자는 투표를 생성할 수 있다.

# 환경변수 설정
.env파일에 MONGODB_URL={각자의 URL 연결}
